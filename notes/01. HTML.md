# Naming Convention

## PascalCase
aka: UpperCamelCase / StudlyCase
应用：react component, js class

## spinal-case 
aka: kebab-case / dash-case
应用：normal-file-name, folder-name, html element
全部小写，🔴不可大写，用 hyphen 连接，最好不要太长（但没有长度限制）。

## camelCase
aka: lowerCamel case / dromedaryCase
应用：JS var, func, method

## snake_case
应用：python / R var name

# Indentation

# HTML Skeleton

- ==href== -> 'style.css'  
- src -> 'script.js'  

```html
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="utf-8">
		<title>child | parent</title>
		<link href="style.css" rel="stylesheet" type="text/css">
		<script src="script.js"></script>
	</head>
	<body>
	</body>
</html>
```

NOTE: `<!DOCTYPE html>` is declaring html5, 🔴no need to add "5"


# Head
🔴all webs must have head & body tags

## meta
- `<meta charset="utf-8">` should always be the 1st tag in head
- meta is 🔴self-closing

## title
- must follow 🔴child-parent pattern, separator can be any special char (except space)
- title should always be the last tag in head❓
- `<title>child | parent</title>`

## link 
- link is 🔴self-closing (aka "stand-alone" tag, in contrast with "paired tags")
- `<link href="style.css" rel="stylesheet" type="text/css">`

## Head Element 顺序

```html
<head>
  <!-- 1. 🔴永远首先加载字符集 Charset -->
  <meta charset="UTF-8">

  <!-- 2. Viewport / SEO -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="This is my website">

  <!-- 3. Title -->
  <title>Child | Parent</title>

  <!-- 4. Stylesheets -->
  <link rel="stylesheet" href="styles.css">

  <!-- 5. 网页图标 Favicon -->
  <link rel="icon" href="favicon.ico">

  <!-- 6. ❌避免 JS scripts 在 head 里，非要放的话必须用 defer/async -->
  <script src="app.js" defer></script>
</head>
```

## Anchor Element
**Link Text Rules**
- 不要用 "click here" 或 过长的 URL, 而要用简短的描述性文本
- 安全性："click here" 或 虚假描述 可能导致被浏览器标记为风险链接
- 无障碍：为了screen reader读起来方便

**Attributes**
- a tag 🔴must have href attribute `href="https://example.com"`
- open in new tab 🔴 `target="_blank"`
- internal anchor `href="#section1"`
- cross-page anchor `href=page.html#id`
- download file `<a href="report.pdf" 🔴download>Download Report</a>`
- mail to `href="mailto:someone@example.com"`
- call tel num `href="tel:+1234567890"`

## Image Element
`<img>` is stand-alone (self-closing) tag
🔴 img 必须有 src 和 alt

```html
<img src="cat.jpg" alt="A cute cat">
```

## List Elements
ul ol --> li (--> img/a)
`<li>` is the ==only direct-child== allowed in `<ul>/<ol>`, `<img>/<a>` should be inside `<li>`

```html
<ul>
  <li><img src="image/apple.png" alt="Apple"></li>
  <li><img src="image/banana.png" alt="Banana"></li>
  <li><img src="image/cherry.png" alt="Cherry"></li>
</ul>
```

## Form Elements

```html
<form action="/submit" method="POST">

  <!-- 文本输入 -->
  <label for="username">用户名：</label>
  <input type="text" id="username" name="username" placeholder="请输入用户名" required>
  <br><br>

  <!-- 密码输入 -->
  <label for="password">密码：</label>
  <input type="password" id="password" name="password" placeholder="请输入密码" required>
  <br><br>

  <!-- 单选按钮 -->
  <p>性别：</p>
  <input type="radio" id="male" name="gender" value="male">
  <label for="male">男</label>
  <input type="radio" id="female" name="gender" value="female">
  <label for="female">女</label>
  <br><br>

  <!-- 复选框 -->
  <p>兴趣：</p>
  <input type="checkbox" id="reading" name="hobby" value="reading">
  <label for="reading">阅读</label>
  <input type="checkbox" id="traveling" name="hobby" value="traveling">
  <label for="traveling">旅行</label>
  <br><br>

  <!-- 提交按钮 -->
  <input type="submit" value="提交">
</form>
```

## Table Elements

```html
<table border="1">
  <thead>
	<tr>
	  <th></th>      <!-- top-left empty cell -->
	  <th>X</th>     <!-- horizontal header -->
	  <th>Y</th>
	  <th>Z</th>
	</tr>
  </thead>
  <tbody>
	<tr>
	  <th scope="row">X</th>  <!-- vertical header -->
	  <td>1.00</td>
	  <td>0.75</td>
	  <td>-0.20</td>
	</tr>
	<tr>
	  <th scope="row">Y</th>
	  <td>0.75</td>
	  <td>1.00</td>
	  <td>0.10</td>
	</tr>
	<tr>
	  <th scope="row">Z</th>
	  <td>-0.20</td>
	  <td>0.10</td>
	  <td>1.00</td>
	</tr>
  </tbody>
</table>
```

visual: 
```
X     Y     Z
X    1.00  0.75 -0.20
Y    0.75  1.00  0.10
Z   -0.20  0.10  1.00
```
