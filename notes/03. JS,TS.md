# OOP: Class

- 通过 `class Car {...}` 声明直接定义的类会有 hoisting;
  但通过 `const Car = class {...}` 定义的类不会被 hoisted。
- `class Car {...}` 内部无需用 `,` 或者 `;` 分隔 methods。
- `this`  --  placeholder variable, aka self (❗️`() => {}` 没有 `this` )
- 

```js
///////////////////////////////////////////////////////////////
// 定义 class
class Car {
	// init
	constructor(name, model, color, year) {
		//🔴fields (properties, attributes)
		this.name = name;
		this.model = model;
		this.color = color;
		this.year = year;
	}
	
	// methods
	turnOn() {
		return `The ${this.color} ${this.model}'s engine is running`;
	}
	
	carAge() {
		const currentYear = new Date().getFullYear();
		return `This car is ${currentYear - this.year} year old`;
	}	
}
```

## Interative Array Methods

`arr.forEach()` `arr.map()` `arr.filter()` `arr.reduce()` 都是 Array methods, 且是 ==High-order Function==，意思是他们把另一个 function 作为 parameter 接收。
Iterative array methods 不会 mutate 原数组。

```js

	/* console: 
		forEach: 1
		forEach: 2
		forEach: 3
	*/


//////////////////////////////////////////////////////////////
// .map()
const doubled = nums.map(n => n*2);
console.log("map:", doubled);
	// map: [2, 4, 6]


//////////////////////////////////////////////////////////////
// .filter()
const evens = nums.filter(n => n % 2 === 0);
console.log("filter:", evens);
	// filter: [2]


//////////////////////////////////////////////////////////////
// .reduce()
const sum = nums.reduce((acc, n) => acc + n, 0);
// acc: accumulator, 用于储存每次操作完后累计的值, 可以叫任意名称
// n: 数组内每个元素
// 0: acc 的 initial value
console.log("reduce:", sum);
	// reduce: 6
```

## Switch
switch 运作方式：找到第一个匹配的 case, 然后==执行以下所有 action, 直到 break==

```js
switch (expression) {
	case value1:    //🔴判断 expression === value1, true 则进入 case
		statement1; //  所有 case 按从前往后顺序判断
		break;      //🔴必须加 break
	case value2:
		statement2;
		break;
	case value3:
		statement3;
		break;
	default:        //🔴如果所有 value 都 !== expression, 则进入 default
		statement0; // default 无论放在哪都是最后一个判断
}
```

# Define Function

## 直接声明

- 用 `function` 关键字声明
- 有 hoisting
- 函数里所有变量都是 block-scope (无论用 let / const / var，都不会 hoist)

```js
// define func
function greet(name) {  // name: parameter
	return `Hello, ${name}!`;  // 这里必须要加 ;
}  //🔴这里不用 ;

// call func
console.log(greet("Alice")); // Hello, Alice!
```

## 定义给变量

- 这里的 `function` 仍然是关键字，不是函数
- 注意：这里greet 是变量名，不是函数本身的名字。赋值给变量为了调试函数
- `function (name)` (可以有空格，function是关键字) 是一个匿名函数
- 没有 hoisting，必须先定义才能调用

```js
const greet = function(name) {  //🔴用 const 定义变量并赋值给变量一个函数
	return `Hello, ${name}!`;
}; //🔴必须写“;”

console.log(greet("Alice")); // Hello, Alice!
```

## 箭头函数

- 箭头函数没有 this，如果写了 this，用的是外层作用域 (module.exports / global / window)
- `{}` 里的是 statement (必须要写 `return` 才能返回值)
- `()` 里的是 expression (会直接返回值，不用写 `return` )，注意用的是==返回值==
- 单行可以省略 expression 的括号，默认是 expression，默认用计算出来的值作为 return

```js
// 通用写法
const varName = (par1, par2) => {
  return expression(par1, par2);  //🔴statement
};

// 单参数省略括号
const varName = par => {
	return expression(par);  // {statement} 必须写 return
};

// 用括号表示 expression
const varName = par => (
	expression(par);  // (expression) 不用写return, 但必须有返回值
);

// 单行省略 return / 省略 expression 括号
const varName = par => expression(par);
```

## Power Calculator (用 recursion / loop 实现)

```html
<!-- index.html -->

<form id="power-form">
	<label for="first-input">Base: </label><input id="first-input">
	<label for="second-input">Exponent: </label><input id="second-input">
	<button type="submit"></button>
</form>

<h1 id="rec-power">
<h1 id="loop-power">
```

```js
// script.js

let recOutput = document.getElementById("rec-output"); // id:spinal-case
let loopOutput = document.getElementById("loop-output");
let form = document.getElementById("power-form");

//🔴 power recurssion function
function powerRec(a, n) {
	if (n===0) return 1;  // recurssion 到 n===0 停止
	return a*powerRec(a, n-1); // a*a*a*a*a*...*1 (共n个a)
};

//🔴 power loop function
function powerLoop(a, n) {
	let result = 1;
	for (let i=0; i<n; i++) {
		result *= a;
	}
	return result;
}

// power form
form.addEventListener("submit", function(e)) {  // e: event obj
		e.preventDefault();  
		// 阻止浏览器默认行为，比如重新加载空表格，自动提交表格等
		
		// get inputs
		let first = document.getElementById("first-input").value;
		let second = document.getElementById("second-input").value;
		
		//🔴convert datatype to num
		first = Number(first);
		second = Number(second);
		
		// cahnge innerHTML
		recOutput.innerHTML=`Recursion: ${powerRec(first, second)}`;
		loopOutput.innerHTML=`Loop: ${powerLoop(first,second)}`;
	}
};
```

## Power Function: Recursion

```js
function myPow() {
	const first = Number(document.getElementById("first").value);
	const second = Number(document.getElementById("second").value);
	
	function recPow(x, n) {
		if (n === 0) {
			return 1;
		} else if (n < 0) {
			return 1 / recPow(x, -n);
		} else {
			return x*recPow(x, n-1);
		}
	}
	
	const result = recPow(first, second);
	
	document.getElementById("output").innerHTML = `
		<h1>${result}</h1>
	`;
}
```
